## Overview

This repository contains certificate(s) awarded by Holberton School France in collaboration with IBM, specifically related to the successful completion of the **Introduction to Open Source** training. It serves as a personal record of learning achievements and as a reference for future employers, collaborators, and anyone interested in open source education pathways.

## About the Certificate

The **Introduction to Open Source** certificate is granted to students who demonstrate foundational knowledge in open source principles, best practices, contribution workflows, and project collaboration. This program, developed with IBM, provides students with the skills needed to participate in open source ecosystems confidently.
